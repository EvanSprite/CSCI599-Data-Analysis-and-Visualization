Project Directory
- TEAM_13_DATAVIS.pdf - Report file
- ufo_sightings-imagecat-index.tar.gz - contains Apache Solr index for imagecat files
- ufo_sightings-noimage-index.tar.gz - contains Apache Solr index for no image files (JSON file)
- dataset Directory - Directory contains all tsv and json datasets.
- SRC Directory - Directory contains all source code and script related files
- imagespace Directory - Directory contains a docker-compose script to initialize all containers related to ImageSpace including ImageSpace, Apache Solr, MongoDB, ElasticSearch, ImageSpace Plugins, and PostgresDB.
- Charts Directory - Includes All charts that we generate using our UFO data
